Objectif

  • Displaying the list of nearby shops sorted by distance
  • Like or dislike a shop
    • If the user likes a shop it will be added to preferred shops
    • If he dislikes a shop it will be disappeared from the home page within 2 hours
  • Displaying the list of liked shops (Preferred shops)
  • Removing a shop from your preferred ones
  • Sign up and login
    • There are two roles : "USER" and "Admin"
    • We can give the ADMIN the access to the panel page
    • for the test => admin : admin@gmail.com/admin1234

Technologies and tools used

  • In the front : I used Angular 8
  • In the back : I used Spring boot
  • For the login: I used Spring Security and Json Web Token (JWT)
